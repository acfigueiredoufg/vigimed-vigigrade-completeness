# vigimed-vigigrade-completeness
Scripts and derived datasets used to assess the completeness of adverse drug event reports in the Brazilian pharmacovigilance system (VigiMed) using the vigiGrade method.

This repository contains the scripts and derived datasets used in the study
"Completeness of adverse drug event reports in the Brazilian pharmacovigilance system using vigiGrade".

## Data source
The analyses were based on secondary data from the Brazilian pharmacovigilance system (VigiMed),
publicly available through the Brazilian Health Regulatory Agency (ANVISA) transparency portal.
Due to regulatory and ethical restrictions, individual-level raw data cannot be redistributed.

## Repository structure
- scripts/: data management and statistical analysis scripts
- data_processed/: derived and aggregated datasets used in the analyses
- results: tables and figures generated by the scripts

## Reproducibility
All data cleaning, variable derivation, and statistical analyses were performed using reproducible scripts.
The repository provides sufficient material to reproduce all analyses presented in the manuscript.

## Reporting guidelines
This study was conducted and reported in accordance with the RECORD-PE guidelines for pharmacoepidemiological studies using routinely collected health data.
